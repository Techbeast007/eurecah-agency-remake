## Installation

1. **Clone the repository:**

    ```bash
    git clone https://github.com/ZUKSKY/eurecah-agency-remake.git
    ```

2. **Navigate to the project directory:**

    ```bash
    cd eurecah-agency-remake
    ```

3. **Install dependencies:**

    ```bash
    npm install
    ```

    [or]

    ```bash
    yarn install
    ```
